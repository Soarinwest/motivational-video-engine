# Völuspá — Anonymous (Poetic Edda)

**Source ID prefix:** `voluspa_`
**Canonical file** for entries from the Völuspá, "Prophecy of the Seeress" — the cosmological poem that opens the Poetic Edda.

## Source metadata defaults

- **Author:** Anonymous (Old Norse, c. 10th-11th century CE)
- **Work:** Völuspá
- **Collection:** Poetic Edda
- **Era written:** `medieval_world`
- **World depicted:** Norse cosmological
- **Region:** `medieval_world`
- **Traditions:** `norse_myth`
- **Default themes:** `fate`, `mortality`, `prophecy`, `duty`
- **Default rights status:** `public_domain_translation` (Bray 1908, Bellows 1923, Thorpe 1866 — public domain)
- **Default visual worlds:** `northern_fate`
- **Default series:** `literary_dark`, `hard_truths_for_men`
- **Translator notes:** Bray, Bellows, and Thorpe are all public domain. Stanza numbers vary.
- **Source URLs:** Wikisource, Project Gutenberg, Heimskringla.no.
